# RestAssuredFrameWork
RestAssuredFrameWork With Maven & TestNG

## Tools & Setup:
- Postman
- IntelliJ
- Java
- RestAssured
- Maven
- TestNG

## Core Features:
- Spotify OAuth2.0
- Register Client App
- Authorize App
- Fetch Access Token and Refresh Token
- Renew Access Token
- Automate Playlist API
- Automate Positive and Negative Scenarios
